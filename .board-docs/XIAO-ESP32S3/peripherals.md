# XIAO ESP32S3 — 板载外设型号表 (Peripherals)

> 来源：https://wiki.seeedstudio.com/xiao_esp32s3_getting_started/  
> 采集日期：2026-06-26

---

## 主控芯片

| 外设类型 | 型号 / 规格 | 接口 | 备注 |
|---------|------------|------|------|
| MCU     | ESP32-S3R8  | -    | Xtensa LX7 双核 32-bit，最高 240 MHz；内置 8MB PSRAM |
| 架构    | Xtensa LX7 dual-core | - | 32-bit |
| 主频    | 最高 240 MHz | - | - |

---

## 存储

| 外设类型 | 型号 / 规格 | 版本差异 | 备注 |
|---------|------------|---------|------|
| Flash（片上） | 8MB     | 标准版 / Sense 版 = 8MB；Plus 版 = 16MB | SPI Flash |
| PSRAM（片上） | 8MB     | 三个版本均为 8MB | 集成在 ESP32-S3R8 芯片内 |
| MicroSD 卡槽 | ≤32GB FAT | 仅 Sense 版有 | SPI 接口，GPIO7/8/9/21 |

---

## 无线通信

| 外设类型 | 规格 | 接口 | 备注 |
|---------|------|------|------|
| Wi-Fi   | 2.4GHz Wi-Fi（802.11 b/g/n） | 内置 | 板载天线 + U.FL 外置天线接口 |
| Bluetooth | BLE 5.0 / Bluetooth Mesh | 内置 | - |
| 天线    | 板载 PCB 天线 + U.FL 连接器 | LNA_IN | 外接天线时信号覆盖 100m+ |

---

## 传感器（仅 Sense 版）

| 外设类型 | 型号 | 接口 | 分辨率 | 备注 |
|---------|------|------|--------|------|
| 摄像头  | OV3660（当前出货） | DVP 并口 + I2C | 2048×1536 (3MP) | 原出货 OV2640 已停产；兼容 OV5640 |
| 摄像头（可选替换） | OV5640 | DVP 并口 + I2C | 2592×1944 (5MP) | 需购买替换件 |
| 摄像头（旧版） | OV2640 | DVP 并口 + I2C | 1600×1200 (2MP) | 已停产 |
| 数字麦克风 | PDM 数字麦克风 | PDM（GPIO41/42） | - | 默认占用 D11/D12 |

---

## 电源管理

| 外设类型 | 型号 / 规格 | 备注 |
|---------|------------|------|
| 电池充电芯片 | 内置（标准版：50mA 快充 / 3.8mA 涓流） | Plus 版：100mA 快充 / 0.9mA 涓流 |
| 电源输入 | USB Type-C 5V / 3.7V 锂电池（JST 接口） | 电池供电时 5V 引脚无电压 |
| 3.3V LDO | 内置，3V3 输出最大 700mA | - |

---

## LED 指示灯

| 外设类型 | 颜色 | GPIO | 说明 |
|---------|------|------|------|
| 用户 LED   | 橙色 | GPIO21 | 低电平点亮 |
| 充电 LED   | 红色 | -      | 充电时闪烁；充满熄灭；无电池时接 USB 亮 30s 后熄 |

---

## 按键

| 外设类型 | GPIO | 功能 |
|---------|------|------|
| BOOT 按键 | GPIO0 | 进入 Bootloader：上电/复位时按住 |
| RESET 按键 | CHIP_PU | 芯片复位 |

## 版本对比总览

| 外设 | XIAO ESP32S3 | XIAO ESP32S3 Sense | XIAO ESP32S3 Plus |
|------|-------------|-------------------|--------------------|
| MCU  | ESP32-S3R8  | ESP32-S3R8        | ESP32-S3R8         |
| Flash | 8MB        | 8MB               | 16MB               |
| PSRAM | 8MB        | 8MB               | 8MB                |
| 摄像头 | ✗          | OV3660（兼容OV5640）| ✗                 |
| 麦克风 | ✗          | PDM 数字麦克风     | ✗                  |
| MicroSD | ✗        | ✓（≤32GB FAT）    | ✗                  |
| 额外 GPIO | ✗      | D11/D12（B2B）    | 更多 GPIO（B2B）   |
| 充电电流 | 50mA     | 50mA              | 100mA              |
| 尺寸 | 21×17.8mm  | 21×17.8×15mm（含扩展板）| 21×17.8mm       |

## 功耗参考（3.8V 供电）

| 工作模式 | 标准版 | Sense 版（含扩展板） | Plus 版 |
|---------|--------|---------------------|---------|
| Modem-sleep | 27mA  | 44mA  | 31.6mA |
| Light-sleep | 2mA   | 5mA   | 2.45mA |
| Deep Sleep  | 14µA  | 3mA   | 33.51µA |
| Wi-Fi Active | 100mA | 110mA | 81mA  |
| BLE Active  | 85mA  | 102mA | 101mA  |
| USB Type-C 正常工作 | 5V@19mA | 5V@38.3mA | 5V@28mA |
